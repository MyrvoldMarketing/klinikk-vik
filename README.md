# Klinikk Vik

Profesjonell one-page nettside for **Klinikk Vik** – kosmetisk klinikk i Oslo, drevet av Tonje Vik.

## Innhold
- Hero med tagline og CTA
- USP-strip
- Behandlingskort
- Om Tonje Vik
- Priser med faner (leppefiller, hud, rynker, filler, fjerning, injeksjon)
- Kundeomtaler
- Kontakt med kart (OpenStreetMap)
- Footer

## Stack
- HTML, CSS (custom), vanilla JavaScript
- Google Fonts: Cormorant Garamond + Inter
- Font Awesome 6 (CDN)

## Kjør lokalt
Åpne `index.html` direkte i nettleser, eller kjør en enkel server:
```bash
python3 -m http.server 5500
```
Åpne deretter http://localhost:5500

## Bilder
Hero- og portrettbilde er placeholders fra Unsplash. Bytt ut URL-ene i
`styles.css` (`.hero-image-frame` og `.portrait`) med egne bilder før lansering.
